# Terrivol Music — Sitio Web

## Cómo publicar en GitHub Pages (5 minutos, gratis)

### Paso 1 — Crear cuenta en GitHub
1. Ve a github.com y crea una cuenta gratis con tu email

### Paso 2 — Crear repositorio
1. Clic en "New repository" (botón verde)
2. Nombre: `terrivolmusic` (exactamente así)
3. Marca "Public"
4. Clic en "Create repository"

### Paso 3 — Subir los archivos
1. En el repositorio recién creado, clic en "uploading an existing file"
2. Arrastra TODOS los archivos de esta carpeta (incluyendo la carpeta artistas/)
3. Escribe "Primer lanzamiento" en el campo de comentario
4. Clic en "Commit changes"

### Paso 4 — Activar GitHub Pages
1. Ve a Settings (arriba a la derecha del repositorio)
2. En el menú izquierdo busca "Pages"
3. En "Source" selecciona "Deploy from a branch"
4. Branch: "main" / Folder: "/ (root)"
5. Clic en Save

### Paso 5 — Tu link listo
En 2-3 minutos tu sitio estará en:
`https://[tunombredeusuario].github.io/terrivolmusic/`

Ese link va en tu bio de Instagram.

## Para editar después sin código
1. Ve a github.com/[tuusuario]/terrivolmusic
2. Clic en cualquier archivo .html
3. Clic en el ícono de lápiz (editar)
4. Cambia el texto que necesites
5. Clic en "Commit changes"
Los cambios se publican solos en minutos.
